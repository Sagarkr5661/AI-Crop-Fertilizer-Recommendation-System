# AI Crop & Fertilizer Recommendation System

**Short:** A Streamlit app that recommends crops (and fertilizer suggestions) based on soil and weather features.  
**Stack:** Python, scikit-learn, Streamlit, Docker, GitHub Actions, Ansible, Nagios.

---

## Features
- Predict crop recommendation using a trained ML model.
- Simple Streamlit UI for input.
- Dockerized for easy deployment.
- CI/CD workflow to build & publish Docker image.
- Ansible playbook for server provisioning & deployment.
- Nagios monitoring sample for app uptime.
